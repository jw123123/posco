# POSCO E&C 웹사이트 리디자인

포스코이앤씨(POSCO E&C) 공식 홈페이지 리디자인 프로젝트입니다.

## 📋 프로젝트 개요

기존 사이트의 고정 뷰포트, 비직관적인 정보 구조, CTA 부재 등의 문제를 개선하고
**Intuitive · Immersive · Smart High-Tech** 컨셉을 기반으로 브랜드 경험을 재설계했습니다.

## 🎨 디자인 컨셉

> 포스코이앤씨의 복잡한 사업 구조를 누구나 쉽게 탐색할 수 있도록 직관적인 정보 구조를 기반으로,
> 풀스크린 영상과 다크 네이비 비주얼로 브랜드에 몰입하는 경험을 제공합니다.
> 각 섹션의 마이크로 인터랙션과 동적 전환 효과를 통해 스마트 하이테크 기업으로서의 정체성을 시각적으로 전달합니다.

## ✅ 주요 개선사항

- **GNB** — 사용자 목적 기반 순서로 재설계
- **히어로** — 풀스크린 배경 영상 및 브랜드 카피 적용
- **사업소개** — 비대칭 이미지 그리드 레이아웃
- **대표실적** — 카드형 슬라이드 구성
- **ESG** — 항공 이미지 배경 오버레이
- **뉴스** — 자동 가로 슬라이드
