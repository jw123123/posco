# 🏗️ POSCO E&C Website Renewal

> 정보 전달을 넘어, 브랜드의 몰입 경험으로

포스코이앤씨 공식 홈페이지를 리디자인한 HTML/CSS/JavaScript 기반 프로젝트입니다. 기존 사이트의 정적인 정보 전달 구조를 개선하고, 친환경 미래 건설 기업의 가치와 방향성을 몰입감 있는 브랜드 경험으로 전달하는 것을 목표로 제작하였습니다.

🔗 배포 사이트: https://jw123123.github.io/posco/

---

## 📖 프로젝트 개요

### 프로젝트 목적

포스코이앤씨는 친환경 미래 인프라 건설 및 스마트 기술을 선도하는 대한민국 대표 종합 건설·엔지니어링 기업입니다.

하지만 기존 홈페이지는 전반적인 UI 구조와 인터랙션이 다소 보수적이고 정보 전달 중심으로 구성되어 있어 브랜드 경험 측면에서 아쉬움이 있었습니다. 이에 '친환경 미래 건설'이라는 브랜드 방향성을 사용자가 직관적이고 몰입감 있게 경험할 수 있도록 UI/UX를 리디자인하였습니다.

### 프로젝트 목표

- 브랜드 아이덴티티를 시각적으로 전달하는 일관된 디자인 시스템 구축
- 직관적인 정보 구조와 네비게이션 설계
- 동적 인터랙션을 통한 브랜드 몰입감 강화
- 모바일·PC 모두에서 편리한 반응형 경험 제공

### 프로젝트 방식

- 개인 프로젝트 (100%)
- 진행 기간: 2026.04.30 ~ 2026.05.11
- Figma를 활용한 UI 설계 및 와이어프레임
- HTML/CSS/JavaScript 기반 퍼블리싱
- Git을 통한 버전 관리

## ✨ 주요 기능

- GNB Hover 시 컬러 전환 및 Depth2 Slide Toggle
- Hero 풀스크린 배경 영상 및 좌하단 브랜드 카피 구현
- 사업소개 영역 이미지 Hover Scale & Radius 효과
- 대표실적 카드 슬라이드(전환 시 카드가 뒤로 넘어가는 애니메이션)
- ESG 카테고리 Hover 시 블러 박스 → 블루 솔리드 전환
- 뉴스 영역 Auto 가로 슬라이드
- 인재채용 영역 Hover 시 Radius·Scale 동시 적용
- Footer 맨 위로 스크롤 버튼
